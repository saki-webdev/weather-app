# Weather App

JavaScriptを使って作成した天気アプリです。

都市名を入力すると、OpenWeather APIから天気情報を取得し、現在の天気を表示します。

## URL



## 機能

* 都市名検索
* 現在の気温表示
* 天気アイコン表示
* 湿度表示
* 風速表示
* 検索履歴保存（localStorage）
* ダークモード切り替え
* Enterキー検索
* ローディング表示
* 天候ごとの背景アニメーション

  * 晴れ：光エフェクト
  * 雨：雨アニメーション
  * 雪：雪アニメーション
  * 曇り：雲アニメーション
  * 雷雨：雷エフェクト

## 使用技術

* HTML
* CSS
* JavaScript
* OpenWeather API
* localStorage

## 工夫した点

* 天候ごとに背景演出を変更し、視覚的に楽しめるようにしました。
* 検索履歴を保存し、再検索しやすくしました。
* ダークモードを実装し、見やすさを向上させました。
